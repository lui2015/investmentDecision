# 债券投资决策表模板

> 适用场景：用于你在买入债券（含可转债、信用债、利率债等）前，系统性审视信用风险、利率风险和流动性风险，避免"只看收益率不看风险"的致命错误。  
> 设计思路综合了格雷厄姆"安全边际至上"、霍华德·马克斯"理解风险溢价与周期"、达里奥"全天候配置与宏观周期"、巴菲特"不懂不做"等框架。  
> **原则：先看能不能还钱，再看能赚多少利息；先判断周期位置，再决定久期长短；高收益 = 高风险，没有例外。**

## 十一、给自己的纪律提醒

- 收益率不是越高越好。高收益率是市场对高风险的定价，不要贪这个便宜。
- 不懂信用分析，就买国债或高等级信用债。不丢人，这是理性。
- 不要用短期资金买长久期债券，利率波动可能让你被迫亏损卖出。
- 分散单一发行人风险，任何一家公司都可能出问题——格雷厄姆。
- 关注现金流，不关注利润。还债靠的是现金，不是账面利润。
- 债券违约前通常有迹可循：评级下调、利差走扩、借新还旧困难、行业舆情恶化。
- 持有到期策略可以忽略价格波动，但不能忽略信用风险。
- 投资债券最大的悲剧：为了多赚 1% 的利息，承担了 100% 的本金损失风险。
