# 期货投资决策表模板

> ⚠️ **风险警示：期货具有杠杆属性，风险远高于股票和债券。亏损可能超过全部本金。在填写本决策表之前，请先确认你是否真的需要参与期货市场。**
>
> 适用场景：用于你在开仓前，强制自己完成一套结构化审查，把"冲动交易"变成"有纪律的决策"。  
> 设计思路综合了达里奥"宏观周期与风险平价"、马克斯"风险第一"、利弗莫尔"趋势跟踪与纪律"、段永平"不做空、不借钱、不做不懂的东西"、芒格"反过来想+避免做蠢事"等框架。  
> **原则：先确认你能亏得起，再谈能赚多少；先设止损，再谈入场；没有计划的交易就是赌博。**

## 二、一票否决项（最严格的过滤，期货容错率极低）

> 参考：段永平"不做不懂的东西"、芒格"别做蠢事比做聪明事更重要"、马克斯"先活下来"、利弗莫尔用血的教训证明：没有纪律的交易者最终会被市场消灭。

## 十一、给自己的纪律提醒

- **每笔交易必须有止损。没有例外。**
- **亏损时绝对不加仓。绝对。**
- 不做隔夜重仓。你睡觉的时候，市场在全球交易。
- 不与市场争论。止损到了就走，别问"为什么"。
- 一天亏损达到限额，关掉软件，离开屏幕。
- 不要因为"手痒"就交易。没有信号就没有交易。
- 赚钱不代表你对了，可能只是运气好。保持敬畏。
- 连续盈利后最危险——过度自信是期货交易者最大的敌人。
- 连续亏损后最需要做的不是"翻本"，而是停下来复盘。
- 永远记住：市场会在你最有信心的时候教你做人。
- 段永平说得对：不做空、不借钱、不做不懂的东西。
