## 一、项目概述

### 1.1 核心定位

本系统是一套**结构化投资决策表工具**。用户在做出任何投资决策前，必须填写对应资产类别的决策表，系统引导用户按固定流程审视关键信息，最终输出一份**可审查、可回溯、可复盘**的投资决策记录。

> **一句话定义：把"拍脑袋"变成"填决策表"，把"我觉得"变成"我验证过"。**

### 1.2 四张决策表的设计原则

| 决策表 | 核心原则 | 投资大咖来源 |
|--------|----------|-------------|
| **股票** | 先过滤再评分，先看风险再看收益，先定计划再谈买卖 | 巴菲特、格雷厄姆、芒格、林奇、马克斯、达里奥、费雪、段永平 |
| **基金** | 先看人再看产品，先算成本再算收益，先定纪律再谈申赎 | 巴菲特、芒格、约翰·博格、马克斯、达里奥 |
| **债券** | 先看能不能还钱再看利息，先判断周期再决定久期，高收益=高风险没有例外 | 格雷厄姆、马克斯、达里奥、巴菲特 |
| **期货** | 先确认亏得起再谈赚多少，先设止损再谈入场，没有计划的交易就是赌博 | 达里奥、马克斯、利弗莫尔、段永平、芒格 |

### 1.3 系统不做什么

1. **不提供投资建议** —— 系统只提供结构化决策框架，不替用户做决定
2. **不提供实时行情** —— 专注决策质量，而非交易速度
3. **不做预测** —— 不预测涨跌，只引导用户系统性思考
4. **不鼓励频繁交易** —— 所有设计导向"想清楚再动手"

```

│  │  阈值: 85/60  │ │  阈值: 85/60  │ │  阈值: 85/60  │ │  否决项×10   │ │

└───────────────────────────────────────────────────────────────────────┘
```

### 2.2 四类决策表通用流程

```
新建决策表 → 选择资产类型(股票/基金/债券/期货)
    ↓
[期货专属] 板块〇：风险确认门槛（5项全部勾选才能继续）
    ↓
板块一：标的基础信息
    ↓
板块二：一票否决项（任一项"否" → 终止 or 写反驳理由继续）
    ↓
板块三：定性分析（8维度×5分 = 满分40分）
    ↓
板块四：定量分析（硬指标 + 结论，满分30分）
    ↓
板块五：估值/定价判断（满分20分）
    ↓
板块六：风险清单 + 最坏情况推演
    ↓
板块七：投资逻辑卡片（3-5条短句 + 反方观点）
    ↓
板块八：买入/交易计划
    ↓
板块九：最终评分与决策（40+30+20+10=100分 → 决策阈值 → 结论）
    ↓
板块十：投后跟踪复盘
    ↓
板块十一：纪律提醒
    ↓
板块十二：极简版快速初筛
```

### 2.3 四类决策表结构对比

| 板块 | 股票 | 基金 | 债券 | 期货 |
|------|------|------|------|------|

| **极简版** | 8个问题 | 8个问题 | 8个问题 | 8个问题 |

## 三、股票投资决策表

> 详见配套模板：`股票投资决策表模板.md`（12大板块完整定义）  
> 系统需 100% 实现模板中的所有板块。

### 3.1 板块一：标的基础信息

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 公司名称 | 文本 | 是 | |
| 股票代码 | 文本 | 是 | |
| 所属行业 | 选择/文本 | 是 | 提供行业选项，可自定义 |
| 跟踪日期 | 日期 | 是 | 默认当天 |
| 当前股价 | 数字 | 是 | |
| 市值 | 数字 | 否 | 单位：亿元 |
| 关注原因 | 长文本 | 是 | |
| 信息来源 | 多选 | 是 | 年报/季报/券商研报/财报会/行业数据/自己整理 |

### 3.2 板块二：一票否决项

系统行为：
- 每个检查项为 是/否 选择，附带备注输入框
- 若任一项选"否"，弹出警告：该项未通过一票否决
- 用户可选择"终止决策"或"填写反驳理由并继续"
- 否决项未全部通过时，最终决策页显示红色警示

| 检查项 | 投资大咖来源 |
|--------|-------------|
| 我能用一句人话讲清这家公司怎么赚钱 | 段永平：不懂不做 |
| 主营业务我看得懂，且在我的能力圈内 | 巴菲特：能力圈 |
| 最近3年没有明显财务造假、重大违规、商誉暴雷等红旗 | 格雷厄姆：安全边际 |
| 公司负债水平没有明显失控 | 财务稳健 |
| 核心逻辑不是靠纯题材、纯故事、纯情绪支撑 | 芒格：别做蠢事 |
| 管理层没有反复损害中小股东利益的记录 | 巴菲特：管理层诚信 |
| 我买它不是因为"已经涨很多"或"别人都在买" | 逆向思维 |

### 3.3 板块三：定性分析（满分40分）

8个维度，每个维度1-5分，参考：巴菲特看护城河，费雪看管理层与成长质量，林奇看业务是否朴素可理解。

| 维度 | 核心问题 |
|------|----------|
| 业务可理解性 | 我是否真正理解产品、客户、盈利模式 |
| 行业空间 | 行业还有没有增长空间，是否长期有需求 |
| 护城河 | 品牌、渠道、成本、网络效应、牌照、转换成本等是否存在 |
| 竞争格局 | 是否处于优势地位，价格战风险高不高 |
| 管理层质量 | 是否诚实、理性、重视股东回报 |
| 资本配置能力 | 赚到的钱能否高效再投资或合理分红回购 |
| 商业模式质量 | 现金流、毛利率、复购、议价能力如何 |
| 抗风险能力 | 遇到周期、政策、原材料波动时是否脆弱 |

### 3.4 板块四：定量分析（满分30分）

11项财务硬指标：营收增长、归母净利润增长、ROE/ROIC、毛利率、净利率、经营现金流、自由现金流、资产负债率、有息负债/利息覆盖、应收账款/存货变化、分红/回购记录。

每项填写：近3年表现 / 当前情况 / 评价(好/中/差) / 备注。

底部四项结论：盈利质量、现金流质量、杠杆风险、是否存在"利润好看但现金流难看"的问题。

### 3.5 板块五：估值判断（满分20分）

字段：当前估值方式、当前估值水平、历史估值分位、对标公司估值、合理价值区间、当前价格相对合理价值、安全边际是否足够。

底部估值一句话：这是一家____的公司，目前价格____，安全边际____。

内置估值计算器辅助。

### 3.6 板块六至十二

- **板块六：风险清单** — 6类风险(行业/经营/政策/估值/管理层/黑天鹅) + 最坏情况推演3问
- **板块七：投资逻辑卡片** — 3-5条短句 + 反方观点
- **板块八：买入计划** — 买入区间/仓位/加减仓条件/止错条件/持有周期/复盘频率
- **板块九：最终评分** — 40+30+20+10=100 → 85/70/60阈值 → 结论
- **板块十：投后跟踪** — 事件记录表 + 复盘四问
- **板块十一：纪律提醒** — 6条纪律（不懂不买/先写理由再下单…）
- **板块十二：极简版** — 8个核心问题快速初筛

## 四、基金投资决策表

> 详见配套模板：`基金投资决策表模板.md`（12大板块完整定义）  
> 核心差异：选基金的核心是选人（主动型）或选规则（指数型），费率是唯一可预测的回报侵蚀因素。

### 4.1 板块一：标的基础信息

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 基金名称 | 文本 | 是 | |
| 基金代码 | 文本 | 是 | |
| 基金类型 | 选择 | 是 | 主动管理/被动指数/ETF/LOF/FOF/QDII/债券型/货币型/混合型 |
| 跟踪指数（如有） | 文本 | 条件必填 | 指数基金必填 |
| 基金经理 | 文本 | 是 | |
| 基金公司 | 文本 | 是 | |
| 成立日期 | 日期 | 是 | |
| 最新规模（亿元） | 数字 | 是 | |
| 当前净值 | 数字 | 是 | |
| 跟踪日期 | 日期 | 是 | 默认当天 |
| 关注原因 | 长文本 | 是 | |
| 信息来源 | 多选 | 是 | 基金季报/年报/基金经理访谈/持仓明细/第三方评级/自己整理 |

### 4.2 板块二：一票否决项

| 检查项 | 投资大咖来源 |
|--------|-------------|
| 我理解这只基金投的是什么（底层资产、行业方向、投资策略） | 巴菲特：不懂不做 |
| 基金经理从业≥5年，且经历过至少一轮完整牛熊周期 | 经验验证 |
| 基金规模适中（主动型不超500亿，不低于2亿；指数型不低于1亿） | 博格：规模与策略容量 |
| 基金没有频繁更换基金经理（近3年内≤1次） | 管理稳定性 |
| 我不是因为最近排名靠前/收益暴涨才关注它 | 芒格：别做蠢事 |
| 综合费率（管理费+托管费+申赎费）我了解且可接受 | 博格：成本决定长期回报 |
| 基金没有严重的合规问题或被监管处罚记录 | 马克斯：先活下来 |

### 4.3 板块三：定性分析（满分40分）

8个维度，参考：博格推崇低成本指数基金，芒格重视管理人品格和能力圈，马克斯关注"第二层思维"。

| 维度 | 核心问题 |
|------|----------|
| 基金经理能力 | 长期业绩是否优秀且稳定（看3-5年），靠能力还是靠运气/风格押注 |
| 投资理念一致性 | 基金经理是否有清晰且一贯的投资框架，说的和做的是否一致 |
| 利益绑定程度 | 基金经理是否有显著自购（百万以上），与持有人利益是否一致 |
| 风格稳定性 | 持仓风格是否漂移，晨星九宫格位置是否跳跃 |
| 下行保护能力 | 在2018/2022等熊市中最大回撤是多少，相对同类排名如何 |
| 持仓质量 | 重仓股是否有护城河，集中度是否合理（前十占比） |
| 费率与成本 | 综合持有成本在同类中是否有竞争力 |
| 基金公司平台 | 基金公司整体投研实力如何，风控体系是否健全 |

### 4.4 板块四：定量分析（满分30分）

8项核心业绩指标（总收益率、年化收益率、超额收益、同类排名、最大回撤、波动率、夏普比率、卡玛比率），按近1/3/5年填写。

5项补充指标：持仓集中度、换手率、机构持有比例、规模变化趋势、跟踪误差（指数基金）。

3项结论：收益来源（能力vs运气）、风险控制、业绩稳定性。

### 4.5 板块五：估值与买入时机（满分20分）

| 字段 | 说明 |
|------|------|
| 底层资产当前估值水平 | PE分位/PB分位/指数点位分位 |
| 历史估值分位 | 低(<30%) / 中(30-70%) / 高(>70%) |
| 市场整体温度 | 偏冷/中性/偏热 |
| 建仓方式 | 一次性买入/分批建仓/定投 |
| 定投计划（如适用） | 频率/每次金额/计划持续时长 |
| 当前时点是否适合大额买入 | 是/否/不确定 |
| 安全边际判断 | 充足/一般/不足 |

底部一句话：这是一只____类型的基金，当前底层资产估值处于____水平，适合____方式介入。

### 4.6 板块六：风险清单

8类专属风险：基金经理离职/更换、规模暴增导致策略失效、风格漂移、底层资产估值过高、行业集中度过高、赎回踩踏、市场系统性下跌、流动性风险(QDII额度/停牌等)。

+ 最坏情况推演3问。

### 4.7 板块七至十二

- **板块七：投资逻辑卡片** — 3-5条 + 反方观点 + 市场对该基金最常见的批评
- **板块八：买入计划** — 买入方式/首笔投入/定投计划/仓位上限/加仓减仓赎回条件/持有周期/复盘频率
- **板块九：最终评分** — 40+30+20+10=100 → 85/70/60阈值 → 结论（动作增加"定投"选项）
- **板块十：投后跟踪** — 事件记录表 + 基金专属复盘5问（经理理念一致性？规模影响？选基正确vs市场行情？）
- **板块十一：纪律提醒** — 8条纪律（不追热门/不频繁申赎/排行榜冠军第二年通常平庸/费率是确定损耗…）
- **板块十二：极简版** — 8个核心问题

## 五、债券投资决策表

### 5.1 板块一：标的基础信息

| 债券名称 | 文本 | 是 | |
| 债券代码 | 文本 | 是 | |
| 债券类型 | 选择 | 是 | 国债/地方政府债/政策性金融债/企业债/公司债/可转债/中期票据/短融/ABS/次级债 |
| 发行人 | 文本 | 是 | |
| 发行人所属行业 | 选择/文本 | 是 | |
| 信用评级（主体/债项） | 选择 | 是 | |
| 评级机构 | 文本 | 否 | |
| 票面利率 | 数字 | 是 | % |
| 付息方式 | 选择 | 是 | 年付/半年付/到期一次付 |
| 发行日 | 日期 | 是 | |
| 到期日 | 日期 | 是 | |
| 剩余期限 | 自动计算 | — | |
| 面值 | 数字 | 是 | |
| 当前价格（净价） | 数字 | 是 | |
| 当前全价 | 数字 | 否 | |
| 到期收益率(YTM) | 数字 | 是 | % |
| 特殊条款 | 文本 | 否 | 回售/赎回/调整票面/转股 |
| 关注原因 | 长文本 | 是 | |
| 信息来源 | 多选 | 是 | 募集说明书/评级报告/发行人财报/债券市场数据/自己整理 |

### 5.2 板块二：一票否决项

| 检查项 | 投资大咖来源 |
|--------|-------------|
| 我理解这只债券的基本条款（票面利率、付息方式、到期日、回售/赎回/转股条款） | 巴菲特：不懂不做 |
| 发行人主体信用评级≥AA（低于AA需有充分信用分析能力） | 格雷厄姆：安全底线 |
| 发行人近3年没有发生过债务违约、技术性违约或重大信用事件 | 信用安全 |
| 发行人利息保障倍数>2（有能力覆盖利息支出） | 偿债能力 |
| 我不是只因为收益率高就想买（高收益=高风险没有例外） | 马克斯：理解风险溢价 |
| 我了解这只债券的流动性状况，不会面临"想卖卖不掉"的困境 | 流动性安全 |
| 我的投资期限与债券剩余期限匹配 | 避免被迫不利时点卖出 |

### 5.3 板块三：定性分析（满分40分）

8个维度，参考：格雷厄姆对债券的核心要求是"偿债能力的充足性"，马克斯强调"理解信用风险溢价的来源"。

| 维度 | 核心问题 |
|------|----------|
| 发行人主业质量 | 主营业务是否稳定、可预期，现金流是否充裕 |
| 行业前景与周期 | 所在行业是否稳定或上行，是否面临结构性衰退 |
| 偿债意愿 | 发行人是否有良好的信用记录和偿债文化 |
| 股东背景与支持 | 是否有强大股东或政府隐性担保 |
| 担保与增信 | 是否有第三方担保、抵质押物或其他有效增信 |
| 条款友好度 | 回售/赎回/调整票面等条款是否对投资者友好 |
| 同类比较优势 | 相比同评级同期限可比债券，综合条件是否更优 |
| 信息透明度 | 发行人信息披露是否充分、及时，财务数据是否可信 |

### 5.4 板块四：定量分析（满分30分）

**发行人财务指标（10项）**：营收规模与稳定性、净利润趋势、经营性现金流、资产负债率、有息负债规模、利息保障倍数（底线>2，安全>4）、现金短债比（底线>1）、速动比率、受限资产比例、对外担保规模。

**债券市场指标（6项）**：YTM、修正久期、凸度、信用利差（对比历史分位）、信用利差变动趋势、日均成交量/金额。

4项结论：短期偿债能力、中长期偿债能力、再融资能力、是否存在"利润不错但现金流紧张"的问题。

### 5.5 板块五：估值与收益率判断（满分20分）

| 字段 | 说明 |
|------|------|
| 当前YTM | % |
| 同评级同期限可比债券YTM | 是否有超额收益 |
| 信用利差历史分位 | 低(<30%)/中/高(>70%) |
| 无风险利率水平 | 国债收益率 |
| 信用风险溢价是否足够补偿违约概率 | 是/否 |
| 利率周期位置 | 加息/降息/平台期 |
| 收益率曲线形态 | 正常(向上)/平坦/倒挂 |
| 久期是否匹配投资期限 | 是/否 |

底部一句话：这是一只____评级的____债券，当前YTM____%，信用利差处于历史____分位，收益率对信用风险的补偿____。

### 5.6 板块六：风险清单

9类专属风险：信用违约、利率上行(价格下跌)、流动性(卖不掉)、评级下调、发行人经营恶化、政策/监管、提前赎回(被叫回)、再投资(到期后收益率下降)、通胀侵蚀。

+ 最坏情况推演3问。

### 5.7 板块七至十二

- **板块七：投资逻辑卡片** — 3-5条 + 反方观点 + "市场为什么给这个收益率"
- **板块八：买入计划** — 买入价格/数量/仓位/单一发行人上限/持有策略(持有到期/交易型/阶梯配置)/加减仓条件/止损条件/复盘频率
- **板块九：最终评分** — 40+30+20+10=100 → 85/70/60阈值 → 结论
- **板块十：投后跟踪** — 事件记录（"是否影响信用安全"替代"是否影响核心逻辑"）+ 债券专属复盘5问
- **板块十一：纪律提醒** — 8条纪律（收益率不是越高越好/不懂信用分析就买高等级/关注现金流不关注利润/分散单一发行人…）
- **板块十二：极简版** — 8个核心问题（发行人能还钱吗？信用评级够安全吗？…）

## 六、期货投资决策表

> 详见配套模板：`期货投资决策表模板.md`（12+1大板块完整定义）  
> **核心差异：新增板块〇"风险确认门槛"，一票否决项最多（10项），决策阈值最严（90分才允许开仓），专设风险收益比计算板块。**

### 6.1 板块〇：风险确认门槛（期货专属，必须全部勾选才能继续）

> ⚠️ 进入期货决策表时，系统必须首先弹出风险确认弹窗。

| 确认项 |
|--------|
| 我使用的是闲置资金，全部亏完不影响我的正常生活 |

| 我做期货不是为了"翻本""赌一把"或"找刺激" |

**全部确认后方可继续填写。任一未勾选 → 页面锁定，不允许继续。**

### 6.2 板块一：标的基础信息

| 一手保证金 | 自动计算 | — | |

### 6.3 板块二：一票否决项（10项，最严格）

> 期货市场容错率极低。任一项为"否"，**必须放弃，无反驳选项**。

| 检查项 | 投资大咖来源 |
|--------|-------------|
| 我真正理解这个品种的供需基本面和定价逻辑（不是只看K线） | 段永平：不懂不做 |
| 我了解该品种的合约规则、交割规则、涨跌停板和保证金调整机制 | 合约理解 |
| 我使用的是闲置资金，没有借钱做期货 | 段永平：不借钱 |
| 本次交易有明确的止损位，且我承诺严格执行 | 利弗莫尔：纪律 |
| 单笔最大亏损≤总资金的2%（绝对上限） | 风控铁律 |
| 总持仓保证金占比<30%（不会因行情波动被强制平仓） | 仓位安全 |
| 不在临近交割月进行投机 | 避免逼仓风险 |
| 我不是因为刚亏了钱想要"翻本" | 赌徒心态检测 |
| 我不是在盘中冲动下决定交易的（经过至少数小时冷静分析） | 芒格：别做蠢事 |
| 今天已有的亏损没有触达日内止损上限 | 日内风控 |

**与其他类型不同：期货否决项不通过时，不提供"写反驳理由继续"的选项，直接终止。**

### 6.4 板块三：定性分析（满分40分）

8个维度，参考：达里奥的宏观框架，利弗莫尔的趋势确认，芒格的多元思维模型。

| 维度 | 核心问题 |
|------|----------|
| 品种理解深度 | 是否理解产业链上下游、成本结构、供需弹性 |
| 供需格局判断 | 当前供需是否有明确的方向性矛盾 |
| 库存与基差信号 | 库存高低、期现基差结构是否支持交易方向 |
| 季节性规律 | 当前时间点是否处于有利的季节性窗口 |
| 政策与外部因素 | 是否有关税、环保、进出口政策等重大变量 |
| 资金面与持仓分析 | 主力资金方向是否明确，持仓量变化是否印证观点 |
| 宏观环境 | 宏观经济周期、通胀/通缩预期、汇率因素是否有利 |
| 技术面确认 | 技术形态是否支持交易方向 |

### 6.5 板块四：定量分析（满分30分）

10项供需数据：现货价格、期现基差(升贴水)、库存水平、库存变化趋势(去库/累库)、开工率/产能利用率、产业链利润、持仓量变化、成交量趋势、主力合约月间价差、相关品种联动。

每项填写：当前数据 / 历史对比 / 评价(利多/中性/利空) / 备注。

结论：供需矛盾是否明确、数据是否支持方向、是否存在数据互相矛盾。

### 6.6 板块五：风险收益比计算（满分20分，期货专属）

> **核心公式：风险收益比 = 目标盈利空间 / 止损空间。最低要求 ≥ 2:1。**

| 字段 | 说明 |
|------|------|
| 计划入场价 | |
| 止损价 | **必填，没有止损=不允许交易** |
| 止损空间（点数） | |
| 止损金额（每手） | |
| 第一目标位 | |
| 目标盈利空间（点数） | |
| 目标盈利金额（每手） | |
| **风险收益比** | 系统自动计算，≥2:1才值得做 |
| 计划开仓手数 | |
| 单笔最大亏损金额 | 必须≤总资金2% |
| 占总资金比例 | % |

**风险收益比<2:1时，系统弹出警告并建议放弃。**

### 6.7 板块六：风险清单

8类专属风险：方向判断错误、突发事件(政策/天灾/地缘)、流动性(涨跌停/临近交割)、保证金上调被迫减仓、逼仓、跳空缺口(隔夜风险)、关联品种联动、**自身情绪失控**。

+ 最坏情况推演3问。

### 6.8 板块七至十二

- **板块七：交易逻辑卡片** — 3-5条 + 反方观点 + "我的判断最脆弱的假设是哪个"
- **板块八：交易计划** — 入场价/入场条件(触发信号)/手数/保证金占比/**止损价+止损动作(无条件执行)**/止盈目标/是否移动止损/加仓条件(**只在盈利方向，绝不在亏损方向**)/最大持仓时间/日内止损上限
- **板块九：最终评分** — 40+30+20+10=100 → **90分才允许开仓/80-89等待/<80放弃**（最严格）
- **板块十：投后跟踪** — 增加"持仓状态"和"盈亏"列 + 平仓后复盘6问（入场逻辑？时机？止损执行？判断vs运气？违反计划？最该改进？）
- **板块十一：纪律提醒** — 11条纪律（止损无例外/亏损绝不加仓/不做隔夜重仓/连续盈利后最危险/连续亏损后停下来复盘/市场会在你最有信心时教你做人…）
- **板块十二：极简版** — 8个核心问题

## 七、公共支撑模块

### 7.1 估值计算器

内置于决策表的估值板块，也可独立使用：

| 方法 | 适用 | 输入参数 | 输出 |
|------|------|----------|------|

| PEG估值法 | 股票 | PE、盈利增速 | PEG值及评级 |
| 格雷厄姆公式 | 股票 | EPS、增长率、AAA债券收益率 | 内在价值 |
| DDM股利折现 | 股票 | 每股股利、折现率、增长率 | 合理价值 |

| 久期/凸度计算 | 债券 | 现金流序列、折现率 | 久期、凸度 |

### 7.2 决策表管理

- **列表视图**：所有决策表按时间倒序，可按资产类型(股票/基金/债券/期货)筛选
- **状态标签**：进行中 / 已完成 / 已放弃 / 持仓中 / 已了结
- **搜索**：按名称、代码、行业搜索
- **排序**：按创建时间、总分、状态排序
- **分数标签**：颜色标记（≥85绿 / 70-84蓝 / 60-69黄 / <60红；期货：≥90绿 / 80-89蓝 / <80红）

### 7.3 导出功能

- 导出为 PDF（打印友好格式，保持决策表完整结构）
- 导出为 Markdown（方便分享和存档）
- 导出为 Excel/CSV

### 7.4 投资知识库

- **大师语录**：按场景分类（买入时/恐惧时/贪婪时/估值时/风控时），在决策表填写过程中上下文推荐
- **认知偏差清单**：8大常见偏差（锚定效应、确认偏差、损失厌恶、从众心理、过度自信、近因偏差、沉没成本、幸存者偏差），附典型表现和应对方法
- **决策表填写指南**：每个板块配有填写说明和示例

### 7.5 数据统计与回顾

- 已完成决策表总数、各类型占比
- 平均评分分布
- 一票否决被触发次数统计（帮助用户认识能力圈边界）
- 投后复盘完成率
- 决策质量回顾：高分决策的后续表现 vs 低分决策的后续表现
