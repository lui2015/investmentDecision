# 基金投资决策表模板

> 适用场景：用于你在申购/买入基金前，系统性审视这只基金是否值得投资，避免追热买冷、盲目跟风。  
> 设计思路综合了巴菲特"不懂不做"、芒格"检查清单+逆向思维"、霍华德·马克斯"周期意识"、约翰·博格"成本决定长期回报"、达里奥"全天候配置"等框架。  
> **原则：先过滤，再评分；先看人，再看产品；先算成本，再算收益；先定纪律，再谈申赎。**

## 三、定性分析（好基金判断）

> 参考：选基金的核心是选人（主动型）或选规则（指数型）。巴菲特推崇低成本指数基金，芒格重视管理人的品格和能力圈，霍华德·马克斯关注管理人是否有"第二层思维"。

评分规则：1 分很差，3 分一般，5 分优秀。

| 维度 | 核心问题 | 分数（1-5） | 备注 |
|---|---|---:|---|
| 基金经理能力 | 长期业绩是否优秀且稳定（看 3-5 年），是靠能力还是靠运气/风格押注 |  |  |
| 投资理念一致性 | 基金经理是否有清晰且一贯的投资框架，说的和做的是否一致（对比季报持仓与公开观点） |  |  |
| 利益绑定程度 | 基金经理是否有显著自购（百万以上），与持有人利益是否一致 |  |  |
| 风格稳定性 | 持仓风格是否漂移（对比历史各期重仓股变化），晨星九宫格位置是否跳跃 |  |  |

4. 我的收益是来自选基正确，还是碰巧赶上了市场行情？

## 十一、给自己的纪律提醒

- 不追热门基金。排行榜冠军第二年通常表现平庸。
- 不频繁申赎。基金投资是中长期行为，不是短线交易工具。
- 不因为短期排名波动而焦虑。3 年以下的排名没有统计意义。
- 关注基金经理的持仓逻辑和框架，而非净值的日常波动。
- 定投纪律：市场越跌越要坚持，而非恐慌停止。
- 不在牛市末期大额追入，不在熊市底部恐慌赎回。
- 费率是确定的损耗，收益是不确定的。便宜的永远比贵的好——博格。
- 如果你无法描述这只基金为什么好，说明你还不够了解它。
